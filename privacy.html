Privacy

Research checks should not become browsing surveillance.

This overview describes the project’s common privacy boundary. Each extension has a platform-specific policy whose terminology and storage behavior match that platform.

Platform-specific policies

  • Google Chrome extension privacy policy
  • Microsoft Edge extension privacy policy
  • Firefox extension privacy policy
  • Safari Web Extension privacy policy

Current MDPI detection

The browser extension processes page content locally to identify MDPI links and citations. Optional NCBI lookups may transmit only validated scholarly identifiers required for resolution. Settings are stored through the browser extension storage API.

Planned integrity lookups

The planned work-level integrity layer may transmit normalized scholarly identifiers such as DOI, PMID, or PMCID only when a lookup is enabled. It must not transmit complete page text, citation text, search queries, full browsing history, account identifiers, or analytics identifiers.

User control

  • Network-assisted lookups have a documented control.
  • Disabling NCBI lookups results in zero related external requests.
  • Coverage reductions caused by disabling lookups are disclosed.
  • Private-browsing behavior remains platform-specific and opt-in where the platform requires it.

Website

The project website contains no analytics, advertising, account system, database, cookies, or remote JavaScript. Cloudflare may process ordinary network and security metadata needed to deliver the site under its infrastructure terms.

Reporting a privacy issue

Use the relevant repository’s private security process for sensitive reports. Do not post private browsing data, access tokens, unpublished manuscripts, or personal information in a public issue.

No product analytics are used on this site.
